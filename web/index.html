SonicTerm

Terminal soundscape engine for Linux systems

View on GitHub

Features

  • • Runs on headless or minimal Linux systems
  • • Handles up to 32 sample tracks in parallel
  • • Per-sample timing windows reduce repetition
  • • Responds to external data sources and text files
  • • Typewriter panel suited to installation copy
  • • Visual modes include color matrix and Game of Life

Quick Install

$ git clone https://github.com/martynasjocius/sonicterm.git
$ pip install -r requirements.txt
$ ./sonicterm.sh
